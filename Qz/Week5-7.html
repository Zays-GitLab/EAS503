  1. What is the difference between deep copy and shallow copy? When does it matter? How are each accomplished?

Shallow Copy stores the references of objects to the original memory address on the other hand deep copy stores copies of the object's value. Shallow Copy can reflect changes made to the new/copied object in the original object since both are pointing to the same memory location; while deep copy doesn't reflect changes made to the new/copied object in the original object.

Return a shallow copy of x.

copy.copy(x)

Return a deep copy of x.

copy.deepcopy(x[, memo])

Shallow copy remembers where the original item is and it will make a change to the original item. Deep copy will clone the original item so modifying the clone won't affect the original.

  1. Explain the difference between class variable and instance variable? How are they accessed? Which variables can be shared between objects and which cannot? Are class variables always shared between objects?

Class variables are defined within the class construction. Because they are owned by the class itself, class variables are shared by all instances of the class. They therefore will generally have the same value for every instance unless you are using the class variable to initialize a variable. They are used and accessed as below:

class TestClass:

    class_variable = 1 # Belongs to the class. Shared by all instances

t1 = TestClass()

t2 = TestClass()

print('\nPrint class_variable')

print('t1.class_variable: ', t1.class_variable)

print('t2.class_variable: ', t2.class_variable)

Instance variables are owned by instances of the class. This means that for each object or instance of a class, the instance variables are different. Unlike class variables, instance variables are defined within methods. They are used and accessed as below:

class TestClass:

    class_variable = 1 # Belongs to the class. Shared by all instances

    def __init__(self, instance_variable):

        self.instance_variable = instance_variable

t1 = TestClass(1)

t2 = TestClass(2)

print('\nPrint instance_variable')

print('t1.instance_variable: ', t1.instance_variable)

print('t2.instance_variable: ', t2.instance_variable)

The init is to initiate the class instance and the iter is to return a cursor or head of this class or object. Lastly, next will take the head or cursor and iterate to the next value until a given condition. This is how to make objects iterable. We can use a while loop next if we have an array of pointers to other arrays. For example:

class MyRange:

    def __init__(self, limit):

        self.limit = limit

    def __iter__(self):

        self.value = 0

        return self

    def __next__(self):

        if self.value < self.limit:

            output = self.value

            self.value += 1

            return output

        else:

            raise StopIteration

i = iter(MyRange(10))
